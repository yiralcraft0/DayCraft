## ✨ Features

- ➕ **Easy Task Creation:** Add new tasks seamlessly via a clean user input field.
- ✅ **Task Management:** Mark tasks as completed using interactive checkboxes.
- 📋 **Minimalist UI:** A clean, uncluttered, and modern card-based interface.
- 💾 **Local Storage Support:** Keeps track of your daily tasks locally without requiring an internet connection.
- ⚡ **Lightweight & Fast:** Optimized performance with minimal resource consumption.
