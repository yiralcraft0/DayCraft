# 🌟 DayCraft - To-Do List App

**Author:** Priyanshu (*YiralcrafT*)

DayCraft is a simple and lightweight Android To-Do List application built using **Java** in Android Studio.
It helps users manage daily tasks efficiently and stay organized.

---

## 📱 Features

- ➕ Add new tasks easily  
- ❌ Delete completed tasks  
- 📋 Simple and clean UI  
- 💾 Local storage support (no internet required)  
- ⚡ Fast and lightweight performance  

---

## 🛠️ Built With

- Java ☕  
- Android Studio 🧩  
- XML (UI Design) 🎨  
- SQLite / Room Database💾  

---
